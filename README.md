# MacOS Setup

Some install scripts for a new macOS machine.
 * Homebrew
 * Java / jenv
 * Clojure
 * Flutter / Dart / fenv
 * Babashka
 * ClojureCLR
 * Datomic
 * Fresh SSH Keys
 * VSCodium
 * Docker
 * Tuple
 * Firefox


### Usage

```bash
curl -O https://raw.githubusercontent.com/brandoncorrea/macos-setup/refs/heads/master/install.sh
chmod +x install.sh
./install.sh
rm install.sh
```
